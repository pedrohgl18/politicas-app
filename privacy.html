Política de Privacidade - Pega ou Passa

Última atualização: 26/12/2025

Bem-vindo ao Pega ou Passa! A sua privacidade é a nossa prioridade. Esta Política de Privacidade explica como coletamos, usamos, compartilhamos e protegemos suas informações ao utilizar nosso aplicativo mobile desenvolvido por Pega ou Passa Team.

1. Informações que Coletamos

Para fornecer a melhor experiência de "match", coletamos as seguintes informações:

  • Dados de Cadastro: Nome, e-mail, data de nascimento e gênero.
  • Dados de Perfil: Fotos, biografia (quebra-gelo), interesses, profissão e signo.
  • Localização: Coletamos sua localização geográfica aproximada (via GPS) para mostrar perfis próximos a você.
  • Dados de Uso: Informações sobre suas interações no app (curtidas, passes, mensagens enviadas).
  • Dados do Dispositivo: Identificadores únicos para envio de notificações push.

2. Como Usamos seus Dados

  • Funcionalidade Principal: Para criar seu perfil e sugerir conexões relevantes.
  • Comunicação: Envio de notificações sobre novos matches e mensagens.
  • Segurança: Monitoramento para prevenir fraudes e comportamentos abusivos.
  • Melhoria do Serviço: Análise de dados (anônimos) para otimizar a performance do app.

3. Uso de Tecnologias de Terceiros e Permissões

Nosso app utiliza o framework Capacitor para interagir com o sistema Android, solicitando as seguintes permissões:

  • Localização: Necessária para o feed de usuários próximos.
  • Câmera/Galeria: Necessária para o upload de fotos de perfil.
  • Notificações: Para alertar sobre atividades no app.

Utilizamos os seguintes serviços de infraestrutura:

  • Supabase: Para autenticação e banco de dados seguro.
  • Cloudflare R2: Para armazenamento seguro de mídias.
  • Firebase (FCM): Para processamento de notificações push.

Segurança: Todos os dados são transmitidos via protocolos seguros (HTTPS) e armazenados seguindo padrões de criptografia de mercado.

4. Compartilhamento de Dados

Não vendemos seus dados para terceiros. Seus dados de perfil (nome, fotos, interesses) são visíveis apenas para outros usuários do app dentro do fluxo de "swipe".

5. Seus Direitos e Retirada de Consentimento

De acordo com a LGPD e GDPR, você tem o direito de:

  • Acessar, corrigir ou excluir seus dados.
  • Retirar seu consentimento a qualquer momento nas configurações do app.
  • Solicitar a exclusão definitiva de sua conta.

Para exercer esses direitos, você pode utilizar as ferramentas dentro do app ou entrar em contato via e-mail: pedrohgl18@gmail.com.

6. Alterações nesta Política

Podemos atualizar esta política periodicamente. Notificaremos você sobre mudanças significativas através do app ou e-mail.
